# Nmap Network Security Lab

## Overview

This project demonstrates how network scanning tools can be used to discover hosts, identify open ports, and detect running services.

The objective of this lab is to understand how attackers and security professionals gather information about systems on a network.

## Tools Used

- Nmap
- Linux terminal
- Basic TCP/IP networking concepts

## Skills Demonstrated

- Network reconnaissance
- Service detection
- Port scanning techniques
- Security risk identification
